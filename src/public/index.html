    SeaBass IDE greets you!

    SeaBass is a text (code) editor for BQ M10 Ubuntu Edition tablet.

    How to start

    1. Put your project files into ~/.local/share/seabass.mikhael/persistent
    2. Click "Reload tree" button
    3. Select file in the Files tree on the left

    Complete feature list & release notes are available on About page.

    SeaBass IDE v0.2.1

    SeaBass is a text (code) editor for BQ M10 Ubuntu Edition tablet.

    Feataures

    • Autosave on typing
    • Zentabs: up to five simultaneously opened tabs. When the limit is exceeded, the eldest is closed
    • Syntax highlighting for the following languages:
      c, c++, css, ejs, go, html, java, javascript, json, jsx, less, php, python, ruby, sass, scss, shell, stylus, svg, swift, typescript
    • Code formatter for the following file formats:
      .html, .js, .json, .css

    Release notes

    This is an early Beta-release, so expect broken features an so on...

    Bug reports & feature requests are very appreciated. If you have any suggestions or something doesn't work as expected, feel free to open issue on project's page on github: (https://github.com/milikhin/seabass) or send email to mikhael@milikhin.name

    Known issues

    Phone layout is not yet implemented, so only BQ M10 is supported.

    Basic file operations (create/delete/copy/rename) are not yet implemeted.

    Hidden files are... hm, well... hidden, so there is no way to open such files in this release.
