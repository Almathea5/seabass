      Seabass greets you!

      Seabass is a text (code) editor for Ubuntu Touch.

      Complete feature list, keybindings and release notes are available on About and Help pages.

      Known issues

      • Phone layout is WIP, so only BQ M10 is fully supported.
      • Text selection and copy/cut/paste operations via touchscreen don't work on Ubuntu Touch. This is, i believe, due to the outdated Ubuntu's platform components in Cordova. Since the click-based Ubuntu Touch is not actively developed now, i don't think the issues will be fixed for 15.04 (unless if someone updates Cordova to use latest platform components, and that would be very welcomed :-).

      Seabass v0.4.1

      Keybindings

      Seabass relies on Codemirror's project "sublime" mode for default keybindings with an addition of a few own shortcuts.

      • Beautify code (if possible) – Ctrl+Alt+B
      • Find – Ctrl+F
      • Find next – Ctrl+G
      • Focus to file-open field – Ctrl+E, Ctrl+O
      • Redo – Ctrl+Y
      • Replace – Ctrl+H
      • Toggle file tree – Ctrl+T
      • Undo – Ctrl+Z

      Usage notes

      There are two versions of the application available:

      1. Standard version from Ubuntu Store (File System access is limited to Seabass's own directory)
      2. Unconfined version from OpenStore/Github (with unrestricted access to user's Home directory)

      Note: Uninstallation of previously installed version is required when switching from standard to unconfined version and back.

      Note: Seabass will use the last directory you've opened using Navigation mode (can be activated on the Settings page) as a file tree Root directory.

      Creating files

      It is now possible to create files and directories using this input field in the header (paths are always considered as relative to project root):

      • foo/bar/qwerty.txt – create qwerty.txt file and all required parent folders (foo and bar) if they are not exist
      • foo/bar/ – create foo/bar directory
      • path/to/.gitignore – It is possble to create/open hidden files

      Seabass v0.4.1

      Seabass is a text (code) editor for Ubuntu Touch.

      Feataures

      • Autosave on typing
      • Zentabs: up to five (three in sidestage mode) simultaneously opened tabs. When the limit is exceeded, the eldest is closed
      • Syntax highlighting for more than 25 languages:
        c, c++, css, go, html, java, javascript, json, jsx, latex, less, markdown, pascal, php, python, ruby, sass, scss, shell, stylus, svg, swift, typescript, xml, yaml
      • Code formatter for the following file formats:
        .html, .js, .jsx, .json, .less, .scss, .css

      Release notes

      This is an early Beta-release, so expect broken features an so on...

      Bug reports & feature requests are very appreciated. If you have any suggestions, want to have support for other languages or something doesn't work as expected, feel free to open issue on project's page on github or send me emails.
