    SeaBass text editor greets you!

    SeaBass is a very basic IDE for BQ M10 Ubuntu Edition tablet.

    How to start

    1. Put your project files into ~/.local/share/me.r5m.seabass/persistent
    2. Click "Reload tree" button
    3. Select file in the Files tree on the left

    Features

    • Autosave on typing
    • Zentabs: up to five open tabs at the same time. When sixth tab is opened, the eldest one is closed
    • Syntax highlighting for the following file formats:
      .html, .js, .json, .css, .ejs, .jsx, .php
    • Code formatter for the following file formats:
      .html, .js, .json, .css

    Release notes

    SeaBass v0.1.0 is an early Beta-release, so expect broken features an so on...

    Bug reports are very appreciated. If you have any suggestions or something doesn't work as expected, feel free to open issue on project's page on github: (https://github.com/milikhin/seabass) or send email to mikhael@milikhin.name

    Known issues

    Basic file operations (create/delete/copy/rename) are not yet implemeted.

    Hidden files are... hm, well... hidden, so there is no way to open such files in this release.
